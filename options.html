Fill-it parameters

Helpers :

targetName :
Targets fieds on "placeholder", "name" attributs and label (if empty => no filter)

targetType :
Targets fields on "type" attribut (text,date,number,password...). Warning : for simplicity "textarea" is considered as input type, so you can filter on "textarea" type

fillerValue :
Sets the pattern that the randomly generated value must follow to fill the field.

By d√©fault input "checkbox" and "radio" are checked randomly. Each "select" prompt will have a random data associated.

Commands :

Auto fill all : CTRL + SHIFT + A

Auto fill one : CTRL + DOUBLE CLICK

Auto fill form : CTRL + SHIFT + F

You can define an area into "form" with "data-filler" attribut (Instance : Ajax forms without "form" tag)
